# Installing Chromeless via Homebrew

Chromeless is now available as a Homebrew cask! Follow these steps to install it on your Mac:

## Step 1: Tap the Repository

First, add Jonathan Berger's tap to your Homebrew:

```bash
brew tap jonathanpberger/tap
```

## Step 2: Install Chromeless

Now install Chromeless:

```bash
brew install --cask jonathanpberger/tap/chromeless
```

That's it! Chromeless should now be installed on your system.

## Updating Chromeless

To update to the latest version:

```bash
brew update
brew upgrade --cask jonathanpberger/tap/chromeless
```

## Uninstalling

If you want to uninstall Chromeless:

```bash
brew uninstall --cask jonathanpberger/tap/chromeless
```

## Troubleshooting

If you encounter any issues:

1. Make sure Homebrew is up to date: `brew update`
2. Try reinstalling: `brew reinstall --cask jonathanpberger/tap/chromeless`
3. Report issues on our GitHub repository
